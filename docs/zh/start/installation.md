请在安装好golang环境后，进入到你到项目目录下

``` bash
> cd {your_project_location}/{project-name}/
```

安装 PowerWeChat sdk

``` bash
> go get -u github.com/ArtisanCloud/PowerWeChat
```


::: tip

* golang版本 :  >=1.16
* golang官网 :  [国内镜像](https://golang.google.cn/dl/)
* goproxy镜像:  [goproxy.cn](https://goproxy.cn/)

:::
